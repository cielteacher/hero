二、实现的功能模块
================================================================================

✓ 麦克纳姆轮速度解算
  - 支持全向运动（vx前后、vy左右、wz旋转）
  - 自动将笛卡尔速度映射到4个电机转速
  - 轮子参数可配置

✓ 功率管理和控制
  - 实时功率计算（基于转速和电流）
  - 智能功率限制（默认200W）
  - 超限时自动按比例降速
  - 功率余量实时显示

✓ PID速度环控制
  - 每个电机独立PID调节
  - 标准位置式PID实现
  - 支持积分抗饱和和死区处理
  - 参数可灵活调整（Kp=20, Ki=0.5, Kd=0）

✓ 在线检测和故障保护
  - 4个电机在线状态监测
  - 通信超时检测（>200ms）
  - 异常电流检测（>30A）
  - 任一故障自动停止底盘

✓ 模式管理
  - CHASSIS_RELAX：放松模式（停止）
  - CHASSIS_NORMAL：普通模式
  - CHASSIS_FOLLOW_GIMBAL：云台跟随
  - CHASSIS_SPIN：旋转模式

四、API接口简览
================================================================================

初始化：
  Chassis_Instance_Typedef *Chassis_Init(void);

绑定电机：
  void Chassis_Bind_Motor(Chassis_Instance_Typedef *chassis, 
                         uint8_t motor_idx, 
                         DJI_Motor_Instance *motor);

主控制接口：
  void Chassis_Control_Update(Chassis_Instance_Typedef *chassis, 
                              float vx, float vy, float wz);

底层执行函数：
  void Chassis_Speed_Resolve(Chassis_Instance_Typedef *chassis);
  void Chassis_Power_Control(Chassis_Instance_Typedef *chassis);
  void Chassis_Force_Control(Chassis_Instance_Typedef *chassis);
  void Chassis_AliveCheck(Chassis_Instance_Typedef *chassis);

模式控制：
  void Chassis_Set_Mode(Chassis_Instance_Typedef *chassis, 
                       Chassis_Mode_Typedef mode);
  void Chassis_Stop(Chassis_Instance_Typedef *chassis);

五、集成步骤
================================================================================

1. 文件放置
   将chassis.h, chassis.c放入User/Chassis/目录

2. 编译配置
   在项目的编译配置中包含User/Chassis/路径

3. 初始化代码示例
   // 初始化电机（4个）
   // 初始化底盘
   Chassis_Instance_Typedef *chassis = Chassis_Init();
   // 绑定电机
   for (int i = 0; i < 4; i++)
       Chassis_Bind_Motor(chassis, i, motors[i]);
   // 设置模式
   Chassis_Set_Mode(chassis, CHASSIS_NORMAL);

4. 控制循环（推荐周期5ms）
   Chassis_Control_Update(chassis, vx, vy, wz);
   Chassis_Power_Control(chassis);
   Chassis_Force_Control(chassis);
   Chassis_AliveCheck(chassis);

八、测试建议
================================================================================

1. 单电机测试
   - 分别测试每个电机是否能正确响应指令
   - 验证CAN通信是否正常

2.速度解算测试
   - 给定vx=1, vy=0, wz=0，检查电机目标转速
   - 给定vx=0, vy=1, wz=0，检查电机目标转速
   - 给定vx=0, vy=0, wz=1，检查旋转是否正确

3. 功率控制测试
   - 在低速下验证功率计算正确
   - 在高速下验证功率限制生效

4. 在线检测测试
   - 断开一个电机的CAN连接，验证离线检测
   - 验证离线时底盘是否停止

5. 集成测试
   - 遥控器控制底盘运动
   - 验证各方向的运动是否正确
   - 验证旋转和移动的组合运动

================================================================================
九、已知限制和注意事项
================================================================================

1. 运动限制
   - 速度范围建议 ±1.0 m/s
   - 旋转速度范围建议 ±π rad/s
   - 超出范围可能导致功率过高

2. 硬件要求
   - 电机必须是DJI系列（3508/3510等）
   - 必须有CAN通信接口
   - 需要DWT定时器支持

3. 实时性要求
   - 控制周期不能超过10ms（推荐5ms）
   - 超时会导致响应延迟

4. 功率限制
   - 智能功率限制会自动降速
   - 可能导致实际运动与指令不符
   - 需要在应用层处理功率缓冲
